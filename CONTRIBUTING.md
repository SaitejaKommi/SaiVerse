# Contributing to SaiVerse

Thank you for your interest in contributing to SaiVerse. This is a personal project that tells my engineering journey through an interactive 3D adventure, but I welcome thoughtful contributions.

## Code of Conduct

Be respectful, constructive, and kind. This is a portfolio game — not a battlefield.

## Development Setup

```bash
git clone https://github.com/SaitejaKommi/Portfolio.git
cd Portfolio
pnpm install
pnpm dev
```

## Commit Convention

We follow [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` — new feature
- `fix:` — bug fix
- `refactor:` — code change that neither fixes nor adds
- `perf:` — performance improvement
- `docs:` — documentation only
- `chore:` — maintenance, tooling, dependencies

## Pull Request Process

1. Ensure your code passes `pnpm typecheck` and `pnpm lint`.
2. Update relevant documentation if your change affects gameplay or architecture.
3. Keep PRs focused on a single concern.
4. Reference any related issues.
